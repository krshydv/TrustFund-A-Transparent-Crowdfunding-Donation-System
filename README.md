# TrustFund – Transparent Crowdfunding Donation System

A full-stack crowdfunding platform focused on transparency and trust.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT
- Payments: Razorpay

## Project Structure
- frontend/ – React application
- trustfund-backend/ – Express API

## How to Run
### Frontend
cd frontend
npm install
npm run dev

### Backend
cd trustfund-backend
npm install
npm start
